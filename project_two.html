FINANCIAL DASHBOARD

BESPOKE INTERACTIVE DASHBOARD USING POWER BI

Use-case

This use-case relates to a Vendor Due Diligence dashboard I performed to assist in the fund raising process of a real-estate Group which provides property sale services in the residential house market in the UK.

The aim of this Dashobard was very much to provide more details regarding the historical track records of the existing funds. I have, hence, produced quite a lot of charts and key trading metrics tables along with the relevant KPIs that are tracked in this market.

From the point of view of a Lender in the property market, this data was very useful in assessing the funds' past performance and determining their strategy for the future.

Solution

I have used the data modelling tool in Power BI to collect, transform, and connect all existing data, before layering in the necessary measures / DAX calculations to design interactive data (before doing any visuals).

The output took the form of a bespoke dynamic Power BI dashboard, securely hosted online, allowing client and bidders to interact with the analysis. Data viewable to bidders could be restricted if desired. The dashboard could be filtered using slicers that could be tailored as needed. The analysis shown was then dynamically updated based on slicer selections. This enabled the user to drill into the data in an engaging way.

By building this dashboard bottom up, the client was able to drill down into the most granular elements of its data and also manage access to it. At the same time, by showing the business' metrics in an interactive way helped to showcase the key selling messages with the interested parties.

Key takeaways

Reporting in Power BI for this assignement represented numerous advantages:

⚡ Easier to manage overall process

The Power BI dashboard was securely hosted, allowing users to interact with the analysis, and enabling the owner (me) to manage access and/or monitor the ongoing interest in the business by measuring the amount of interaction with the dashboard.

⚡ Better quality

By using Power BI visuals, I've been able to quickly produce high standard analysis which are often difficult to do with Excel e.g. map a customer listing or predict forecast numbers (see demo below), etc.

⚡ Deeper insights

A dynamic dashboard is an effective way to unlock valuable insights hidden within the data. By building dashboards bottom up, the user is able, in just a few clicks, to cut and dice datasets and mine the data to work out what the drivers and correlations really are.

⚡ New way to interact

Need to change a chart or tweak some data in my visuals? Do I need to isolate one customer to get a clearer view of the business trend? Power BI report changes the way you interect with your client. On this assignement I've discussed the key findings with my client, and I had the ability to adjust any analysis on a live basis (without altering the raw data).

Output/Demo

We can slice the data in many different ways by selecting part of a visualization. Any interaction with the data will automatically update the key metrics in this Dashboard!

Power BI contains amazing visualization tools that will instantly elevate any analyis. In this demo, I have used the map visualization so that anyone could directly see how the location affected the deals' performance. Stacking multiple correlation charts is also a great way to reveal what are the key drivers of the profitability.

The forecast feature of this visualization is a powerful tool as it is always tricky to predict future numbers. This forecast tool can be set depending of the confidence %, and gives you an average, upper, and lower bound. The underlying data can then be extracted in a table, if necessary!

I always find it useful to get the detailed analysis in a table so anyone could drill down into the most granular elements of the data and directly spot trends and/or explanations to unsual movements.
